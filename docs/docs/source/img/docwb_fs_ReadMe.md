# Installation

1.Please use below extension in VS code to create/update .wsd files

https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml


2.Open your .wsd file in VS Code.
3.Press Ctrl + Shift + P 
4.Search for PlantUML: Export Current Diagram and select it.
5.Choose the format (e.g., PNG).
The .png file will be saved in the same directory as your .wsd file.
