# Infosys Document Workbench - Guide


The **Infosys Document Workbench** guide contains the detailed instructions for the solution installation. 
Please follow the steps given below to setup the guide locally.

## Installing the Guide

Clone this repository to a local system. 

```
cd c:\
git clone https://github.com/Infosys/Infosys-Document-Workbench.git InfyDocWb
```

Open a terminal window from current location in local system and run the following script file. 
A menu will be shown.

```dos
cd C:\InfyDocWb\docs
.\docs_manager.bat
```

From the menu, select `5. Quick start (1 + 2 + 3)`. 

> This will do all the necessary installations.

## Running the Guide

If not already done from previous step, open a terminal window from current location in local system and run the following script file. 
A menu will be shown.

```dos
docs_manager.bat
```

From the menu, select `11. View documentation`. 

> This will launch the guide in a new browser tab/window.

Follow rest of the instructions from the documentation site.
